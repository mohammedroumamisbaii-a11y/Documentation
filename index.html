The "Zero-to-Dashboard" Pipeline

Objective: Extract Meta Ads Data → SomoCRM MySQL → Metabase Visualization

Phase 01

Facebook Developers Setup

  1. Create App: Go to Meta for Developers. Click Create App > Other > Business.
  2. Add Marketing API: Inside your App Dashboard, find "Marketing API" and click Set Up.
  3. Generate Access Token:

    Navigate to Marketing API > Tools. Select all 3 permissions:

    ads_read ads_management read_insights

    Click "Generate Token" and SAVE it immediately.

  4. Find Ad Account ID: Go to your Meta Ads Manager. The ID is in the URL (usually starts with act_).
Phase 02

Airbyte Installation (Docker)

On your MSI Thin 15, open your terminal (PowerShell or Bash) and run:

# Clone and start Airbyte
git clone https://github.com/airbytehq/airbyte-platform.git
cd airbyte-platform
./run-ab-platform.sh
After setup, open http://localhost:8000. Default login is usually airbyte / password.

The Connection Sync

A. Set Source (Facebook)

In Airbyte Dashboard, click Sources > + New Source. Search for "Facebook Ads". Paste your Access Token and Ad Account ID. Click Set Up Source.

B. Set Destination (MySQL)

Click Destinations > + New Destination. Search for "MySQL".

  • Host: host.docker.internal (Crucial for local Docker DB)
  • Port: 3306
  • Database: somocrm_backend
  • User/Password: Your local MySQL credentials.
Phase 03

Metabase Visualization

Metabase will sit on top of your DB to show the charts.

docker run -d -p 3000:3000 --name metabase metabase/metabase
  1. 1. Visit http://localhost:3000.
  2. 2. Connect to your MySQL database (same credentials as Step 3).
  3. 3. Locate the table airbyte_raw_fb_ads_insights.
  4. 4. Create "Questions" (e.g., "Total Spend per Day") and save them to a SomoCRM Dashboard.
