01. What is Airbyte?

Airbyte is an open-source **Data Integration (ELT)** platform. In our stack, it's the "Truck" that goes to Facebook Ads API, picks up the data, and delivers it to our MySQL database.

Connectors

Access 300+ pre-built connectors for Facebook, Google Ads, Shopify, etc.

Sync Scheduling

Automatically sync data every hour, day, or even every 5 minutes.

02. The Data Pipeline Logic

"How data moves from Meta to our Dashboard"

1

Extract: Airbyte calls the Meta Marketing API using your Access Token.

2

Normalize: Airbyte converts the complex JSON from Meta into a readable SQL format.

3

Load: The data is pushed into our somocrm_db into "Airbyte Raw" tables.

03. Installation (Docker-Compose)

For the MSI Thin 15, we use Docker to run the entire Airbyte ecosystem (Server, Worker, UI).

Terminal Deployment 
# Clone official Airbyte
git clone https://github.com/airbytehq/airbyte-platform.git
cd airbyte-platform

# Run the bootstrapper script
./run-ab-platform.sh

# Access UI at: http://localhost:8000
# User: airbyte / Pass: password

04. Source & Destination Config

MySQL Destination
