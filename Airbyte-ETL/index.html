01. What is Airbyte?

Airbyte is an open-source **Data Integration (ELT)** platform. It's the "Truck" that goes to the Meta Ads API, picks up data, and delivers it to our MySQL.

Connectors

Access 300+ pre-built connectors for Facebook, Google Ads, Shopify, etc.

Sync Scheduling

Automatically sync data every hour, day, or even every 5 minutes.

02. The Data Pipeline Logic

"How data moves from Meta to our Dashboard"

1

Extract: Airbyte pulls raw data via API using your Access Token.

2

Normalize: Airbyte converts complex JSON into clean SQL rows.

3

Load: Data is pushed into somocrm_db as "Airbyte Raw" tables.
