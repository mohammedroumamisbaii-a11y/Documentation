Metabase & Laravel Integration

Step-by-Step Technical Guide for SomoCRM Developers.

Phase 1: Deploy Metabase

Run Metabase using Docker. Ensure it's in the same network as your MySQL database.

docker run -d -p 3000:3000 --name metabase metabase/metabase

Open http://localhost:3000, complete the setup, and connect your somocrm_db.

Phase 2: Security & Signed URLs

To show charts safely inside our CRM, we use Signed Embedding (JWT).

  1. Go to Admin Settings > Embedding.
  2. Click Enable and copy the Embedding Secret Key.
  3. Add this key to your Laravel .env:
METABASE_SITE_URL="http://localhost:3000"
METABASE_SECRET_KEY="your_copied_secret_key"

Phase 3: Generating the Token

In your Laravel Controller, generate a secure JWT token for the specific dashboard ID.

use Firebase\JWT\JWT;

public function getDashboardUrl($id) {
    $payload = [
        "resource" => ["dashboard" => $id],
        "params" => (object)[],
        "exp" => time() + (60 * 10) // 10 mins expiry
    ];
    $token = JWT::encode($payload, env('METABASE_SECRET_KEY'), 'HS256');
    return env('METABASE_SITE_URL') . "/embed/dashboard/" . $token . "#bordered=false&titled=true";
}

Phase 4: Display in Vue.js

Finally, use an <iframe> to show the chart in your Vue dashboard.

<template>
  <div class="dashboard-container">
    <iframe :src="metabaseUrl" frameborder="0" width="100%" height="600" allowtransparency></iframe>
  </div>
</template>
