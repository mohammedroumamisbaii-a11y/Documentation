Concept 01

What is Metabase?

Metabase is the industry's leading open-source Business Intelligence tool. It allows teams to visualize data without writing complex SQL code. By connecting Metabase to your production or data-warehouse database, you turn "Dry Rows" into "Visual Insights."

Visual Querying

Filter and summarize data using a simple GUI—no SQL required.

X-Ray Insights

Automatically generate dashboards based on your table structure.

Native Embedding

Integrate your charts into any web app via secure JWT tokens.

Concept 02

Deployment & Infrastructure

Professional deployment requires isolation. Running Metabase as a Docker container ensures high availability and easy scaling.

# Pull and Start the BI Service
docker run -d -p 3000:3000 --name bi_engine metabase/metabase
Concept 03

Securing Embedded Data

When you embed a dashboard, you must ensure data privacy. Metabase uses Signed Embedding. This means your application signs a request using a Secret Key to verify the user's identity.

Technical Workflow:

  1. Backend generates a JWT with the Dashboard ID and Expiry Time.
  2. Backend signs the JWT using the HS256 algorithm.
  3. Frontend receives the token and loads the Metabase iFrame.
Concept 04

Frontend Implementation

The user sees a native part of your application. The iFrame is completely transparent and responsive.

<iframe src="https://stats.your-app.com/embed/dashboard/TOKEN" width="100%" height="600" allowtransparency></iframe>
